Информация о системе
=================================

**Использование**:

    chmod +x system-info.sh
    ./system-info.sh
    
**Показывает**: 

 - Список последних входивших пользователей
 - Количество процессов
 - Статусы сервисов
 - IP-адреса сетевых интерфейсов
 - Информацию RAM
 - Информацию об устройствах
